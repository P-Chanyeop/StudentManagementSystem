# 학원 관리 시스템 데모

이 프로젝트는 학원 관리 시스템의 데모 버전입니다.

## 주요 기능

### 1. 수업 예약 페이지
- 네이버 예약과 유사한 캘린더 형식
- 날짜, 시간 선택 후 예약자 정보 입력
- 어머니 이름, 연락처와 학생 이름, 연락처 입력

### 2. 출석부 페이지
- 실시간 출석 체크 기능
- 등원/하원 시간 관리
- 수업 진행률 표시
- 출석 통계 및 정렬 기능
- 학생별 상세 정보 모달

### 3. 학생 상세 페이지
- 학생 기본 정보 관리
- 수업 정보 및 잔여 횟수
- 상담 이력 저장 및 조회
- 출석 통계

## 디자인 특징
- 화이트/그린 컬러 테마
- 반응형 디자인 (모바일/태블릿/PC 호환)
- 직관적인 사용자 인터페이스

## 기술 스택
- HTML5
- CSS3 (Flexbox, Grid)
- Vanilla JavaScript
- 더미 데이터 사용

## 데모 사이트
GitHub Pages를 통해 배포된 데모 사이트에서 확인할 수 있습니다.
